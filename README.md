# 🎨 Modern Art Online

在线版 Reiner Knizia 经典拍卖桌游《现代艺术》。3-5 人实时对战，通过拍卖买卖画作，4 回合后现金最多者获胜。

## 快速开始

```bash
git clone <repo-url>
cd modern-art
npm install
npm start        # 或双击 start.bat
```

浏览器访问 `http://localhost:3000`，创建房间后把邀请链接发给朋友即可开局。

> 内网穿透：在 ngrok 等工具的公网地址后，点击大厅「🌐 公共地址」填入即可生成公网邀请链接。

## 游戏规则

### 目标
玩家扮演画商，拍卖五位现代艺术家的画作。共 4 个回合，现金最多者胜。

### 发牌

| 人数 | 初始手牌 | 每回合补牌 |
|------|---------|-----------|
| 3 人 | 11 张 | 4 张 |
| 4 人 | 8 张 | 3 张 |
| 5 人 | 8 张 | 2 张 |

70 张牌在 4 回合中全部分完。

### 五位画家

| 画家 | 颜色 | 数量 |
|------|------|------|
| Lite Metal | 🟡 金色 | 12 |
| Yoko | 🟢 绿色 | 13 |
| Christin P | 🔴 红色 | 14 |
| Karl Gitter | 🔵 蓝色 | 15 |
| Krypto | 🟤 棕色 | 16 |

### 五种拍卖方式

| 图标 | 类型 | 说明 |
|------|------|------|
| 🔊 | 公开竞价 | 自由出价，每次出价重置 15s，计时结束最高者得 |
| ↻ | 一圈价 | 画主左手开始依次出价/放弃，画主最后决策 |
| ✉️ | 暗标 | 所有人秘密出价，全部提交后揭晓 |
| 💰 | 一口价 | 画主定价，依次购买/放弃，无人买则画主自购 |
| 👥 | 联合拍卖 | 打出 + 搭配同画家另一张，按第二张方式拍卖 |

### 计分

每回合某画家出现第 5 张时结束。按出现数量排名：

- 🥇 第 1 名 **+30 万**
- 🥈 第 2 名 **+20 万**
- 🥉 第 3 名 **+10 万**

回合结束时所有已购画作按累计价值卖出，画作清空。

## 技术栈

- **后端**：Node.js + Express + Socket.IO
- **前端**：原生 HTML/CSS/JS，无框架
- **通信**：WebSocket 实时双向

## 功能

- 🏠 创建/加入房间，自定义房间号
- 🔗 邀请链接一键复制，`?room=CODE` 直达
- ⏱️ 90 秒回合计时 + 拍卖倒计时，服务端同步
- 🔄 断线重连（同名接管），观战模式
- 💬 局内聊天
- 🎨 手牌按颜色整理
- 📊 回合结算详细统计（每人收藏明细 + 收益）
- 📖 内置规则文档
- 📱 手机/平板/桌面全响应式
